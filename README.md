# 🏠 Smart Hostel Complaint Management System

A modern web-based application developed to simplify hostel complaint registration and management. The system enables students to submit complaints digitally while allowing hostel administrators to efficiently monitor, assign, update, and resolve complaints through an intuitive dashboard.

---

## 📖 Project Overview

The Smart Hostel Complaint Management System replaces the traditional manual complaint register with an automated digital platform. Students can report hostel-related issues online, attach supporting images, and track complaint status in real time. Administrators can manage complaints, update their progress, and maintain transparency throughout the complaint resolution process.

---

## ✨ Features

### 👨‍🎓 Student Module
- Student Login
- Register Hostel Complaints
- Upload Complaint Images
- View Complaint History
- Track Complaint Status
- Edit Complaint Details (before processing)

### 👨‍💼 Admin Module
- Secure Admin Login
- View All Complaints
- Filter Complaints by Status
- Update Complaint Status
- Delete Invalid Complaints
- Manage Student Records
- Complaint Resolution Dashboard

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Sibidharan-csbs/smart_hostel_complaints.git
```

### Navigate to Project

```bash
cd smart_hostel_complaints
```

### Build Project

```bash
mvn clean install
```

### Run Application

```bash
mvn spring-boot:run
```

Open your browser:

```
http://localhost:8080
```
